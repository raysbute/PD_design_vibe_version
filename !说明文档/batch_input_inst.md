批量导入功能在程序主菜单即可访问，支持两种方式：

使用方式
1. 从文件导入
在程序主菜单选择 "4. 批量数据导入"，然后选择 "1. 从文件"，输入 txt 文件名即可。

文件格式：每行一条记录，用 | 分隔字段，# 开头的行为注释。


# 格式示例
DEPT|心血管内科|诊治心血管疾病
DOCTOR|王医生|0|DEPT001|1,3,5
PATIENT|张三|30|13800001111|0
MEDICINE|阿莫西林|阿莫西林|阿莫仙|0.25g*24|500
WARD|0|DEPT001|30
ADMIN|user001|mypassword
REG|PAT001|DEPT001|DOC001
DEPTMED|DEPT001|MED001
各实体格式说明：

类型	格式	说明
DEPT	名称|描述	ID 自动生成
DOCTOR	姓名|级别|科室ID|出诊星期	级别: 0=主任/1=副主任/2=主治/3=住院；出诊星期用逗号分隔1-7
PATIENT	姓名|年龄|联系方式|类型	类型: 0=门诊, 1=住院
MEDICINE	商品名|通用名|别名|规格|库存量	别名可为空
WARD	类型|科室ID|床位数	类型: 0=普通, 1=特殊
ADMIN	管理员ID|密码	会检查ID是否重复
REG	患者ID|科室ID|医生ID	使用当前 day 生成挂号
DEPTMED	科室ID|药品ID	建立科室-药品关联
2. 从键盘标准输入
选择 "2. 从键盘标准输入"，然后逐行输入，格式同上。输入空行结束。


=== 批量输入模式 ===
格式: ENTITY|field1|field2|...
输入空行结束输入。
DEPT|皮肤科|诊治皮肤疾病
DOCTOR|李医生|2|DEPT006|2,4,6
MEDICINE|红霉素软膏|红霉素||10g|200
[空行回车结束]
示例文件
创建一个 batch_test.txt：


# 添加一个科室
DEPT|康复科|提供康复治疗服务

# 添加两个医生
DOCTOR|赵康复|0|DEPT006|1,3,5
DOCTOR|钱理疗|2|DEPT006|2,4

# 添加药品
MEDICINE|布洛芬缓释胶囊|布洛芬|芬必得|0.3g*20粒|400

# 建立关联
DEPTMED|DEPT006|MED026
然后在程序中选择 "4 → 1 → batch_test.txt" 即可批量导入。
