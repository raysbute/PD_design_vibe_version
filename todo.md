## 进一步修改的要求

### 1. 基本要求

1. 增加用txt文件进行输入的功能，支持通过txt或者键盘输入批量新增数据（可以选择使用哪一种）

### 2. 修复已有bug

### 3. 使用Qt设计一个图形化界面
要求：
1. 管理员界面
左侧菜单：科室管理、医生管理、患者管理、挂号管理、住院管理、药品管理、报表。
科室管理面板
    上方：表格展示所有科室（科室ID、名称、描述、关联病房数、医生数）。
    右侧：按钮 [新增] [修改] [删除]。
    双击科室行 → 弹出“科室详情”对话框，内嵌两个子标签：
        关联病房：表格显示该科室下的病房列表，可拖拽分配/取消。
        关联药品：表格显示全院药品，勾选即可分配。

医生管理面板
    表格：医生ID、姓名、级别、所属科室、出诊星期（如“周一,三,五”）。
    筛选器：按科室下拉框快速过滤。
    按钮：[新增] [修改] [删除] [请假/离职标记]。
    新增/修改时弹出表单：包含科室下拉选择、出诊星期多选框（周一~周日）。

患者管理面板
    表格：患者ID、姓名、年龄、类型（门诊/住院）、联系方式。
    支持按ID或姓名搜索。
    按钮：[新增] [修改] [删除]。
    双击患者行 → 弹出“患者详情”对话框，显示其所有诊疗记录（可再嵌套表格）。

挂号管理面板
    上方：科室选择 → 医生选择（自动过滤当日出诊医生） → 患者选择（支持模糊搜索）。
    实时显示：当前选中医生今日已挂人数/限号数（如20），以及该患者今日是否已挂该科室。
    按钮：[挂号] → 生成挂号记录，更新列表。
    下方：当日挂号列表（表格），可手动将状态改为“已就诊”或“作废”。

住院管理面板
    左侧：病房-床位可视化区域（用网格表示病床，绿色=空闲，红色=占用）。
    右侧：选择病房 → 选择空闲床位 → 选择患者 → 指定责任医生（多选列表） → 输入押金金额 → 点击[办理住院]。
    下方：住院患者列表，双击可查看详情或办理出院。
    出院操作：点击[出院] → 系统计算费用，弹出结算对话框显示“扣除费用：XXX 元，退还押金：XXX 元”，管理员确认后床位释放。

药品管理面板
    药品列表表格：ID、商品名、通用名、库存量、消耗量。
    按钮：[新增] [修改] [删除] [入库] [出库]。
    入库/出库：弹出输入数量对话框，出库时校验库存，更新库存和消耗量。

报表面板

    采用标签页组织：
        药品报表：表格，可按剩余量/消耗量/名称排序，支持科室筛选、搜索。
        医生报表：表格，按ID排序，统计接诊量和处方数（可选时间范围）。
        床位报表：按病房显示床位占用图。
        营业额报表：显示一段时间内挂号费、药费、住院费分项及总计，可绘制简单柱状图（Qt Charts 或手动绘制）。
        医生繁忙度：表格，列出每位医生的接诊数、处方数、负责住院患者数。

2. 医生界面

左侧菜单：我的待诊、门诊工作、我的住院患者、处方与记录查询。
我的待诊面板
    列表显示所有挂号到本人且状态为“待就诊”的患者（显示挂号时间、患者姓名、年龄）。
    按钮：[接诊] → 弹出“看诊对话框”。
    看诊对话框：
        自动带出患者基本信息。
        输入栏：主诉（多行文本）、诊断（多行文本）。
        下方按钮：[开检查] [开处方] [完成看诊]。
        开检查：弹出小窗，输入检查名称、费用 → 生成检查记录。
        开处方：弹出药品选择对话框，左侧药品列表（可搜索），右侧已选药品清单，双击药品可设置数量，确认后生成处方记录和待缴费金额。
        完成看诊：保存看诊记录，挂号状态改为“已就诊”。

我的住院患者面板
    表格展示所有责任医生包含本人的在院患者。
    双击可查看该患者诊疗记录、处方，并可为其开检查/开处方（交互同门诊）。

处方与记录查询面板
    可搜索特定患者，查看历史看诊记录和处方详情。

3. 患者界面

左侧菜单：我的记录、费用支付、住院信息（仅住院患者可见）。
我的记录面板
    树形结构或表格：按时间倒序列出所有挂号→看诊→检查→处方记录，每个条目可展开查看详情。
    每个记录旁显示状态（有效/已撤销）。

费用支付面板
    表格显示所有“待支付”项目（挂号费/处方药费），每行显示项目名称、金额、状态。
    勾选一条或多条 → 点击[支付] → 从患者账户余额扣除（此处可设计为模拟支付，直接改变 money）。
    支付成功后，处方对应的药品库存自动扣减。

住院信息面板（仅住院患者）
    显示当前病房、床位号、责任医生列表、入院天数。
    押金余额实时显示，每日扣费明细以表格列出。
    可在此处[补缴押金]以应对余额不足。

四、交互细节与用户体验设计

    数据校验友好提示：所有不合法输入（如挂号已满、药品库存不足、床位占用）都通过消息框（QMessageBox）给出明确中文提示，禁止继续操作。
    链表视图刷新：每次增删改操作后，表格和列表立即刷新，可采用“重新遍历链表生成模型”的简单方式。

    搜索与过滤：
        大部分表格支持实时搜索框（如 QLineEdit + QCompleter 或输入后点击搜索）。
        过滤常采用顶部下拉框（科室、类型等）。

    录入辅助：
        ID 自动生成建议（如 R-250508-001），用户可修改但不得重复。
        日期选择使用日历控件（QDateEdit），自动转换为 timeDay。
        药品选择时显示库存量，方便决定数量。

    快捷键：常用操作绑定快捷键（Ctrl+N 新增、Ctrl+S 保存、Del 删除等）。
    数据持久化：在“文件”菜单中提供 [保存到文件] [从文件加载]，程序关闭时自动保存。
